# TUI Dashboard Subdomain [SUB-005]

The TUI (Text-based User Interface) Dashboard Subdomain is concerned with providing a rich, interactive terminal-based interface for monitoring and controlling the ColdVox application. It is the primary user interface for the application in its current state.

Key capabilities include:
- **Real-time Status Panels**: Displaying live information about VAD state, STT events, and system metrics.
- **Keyboard Controls**: Allowing the user to control the application using keyboard shortcuts (e.g., Start, Stop, Mute).
- **Log Viewing**: Providing a panel to view real-time application logs.
- **Low Resource Usage**: Offering a lightweight interface that can run in any standard terminal without requiring a graphical environment.
